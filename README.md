نخرج فين؟ 🎬🍔🎮

Smart outing planner that helps users discover the best places to go based on their budget, available time, group size, and city.

Built with React.js.

✨ Features
🎬 Browse cinemas
🍔 Discover restaurants
🎮 Explore activities
💰 Budget-based recommendations
⏰ Time-aware planning
👥 Group size filtering
🏙️ City filtering (Cairo / Giza / Alexandria)
📸 Dynamic place images
📖 "View More" system for cleaner UI
🧠 Smart itinerary generation
🛠️ Tech Stack
React.js
JavaScript (ES6)
CSS3
Vite
